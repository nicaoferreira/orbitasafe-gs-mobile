# 🛰️ OrbitaSafe

## Descrição da Solução

O OrbitaSafe é um aplicativo mobile desenvolvido em React Native com Expo que utiliza o conceito de monitoramento ambiental baseado em dados orbitais para auxiliar na prevenção de desastres naturais.

A solução permite o gerenciamento de áreas monitoradas, visualização de alertas ambientais, acompanhamento de indicadores orbitais e consulta de informações estratégicas relacionadas ao monitoramento de riscos ambientais.

O objetivo do projeto é demonstrar como tecnologias derivadas da exploração espacial, como satélites de observação da Terra e inteligência artificial, podem ser utilizadas para apoiar órgãos públicos e equipes de emergência na tomada de decisões.

---

## Funcionalidades

- Login de usuário
- Cadastro de usuário
- Dashboard com indicadores do sistema
- Gerenciamento de áreas monitoradas
- Exclusão de áreas monitoradas
- Visualização de alertas ambientais
- Consulta de dados orbitais
- Perfil do usuário
- Persistência de dados utilizando AsyncStorage
